Humble beginnings

September 7, 2019

Honestly I have no idea what to write here. Not the most auspicious start for a brand new blog...

So why start a blog if I have nothing to say?

Well, right now I am learning the basics of web development, HTML and CSS, and at this stage I don't feel I have anything to add to the conversation. I do, however, want to write some code and put into practice what I learn along the way. Hopefully, content will develop with further knowledge and experience.

My main educational resources are MDN and freeCodeCamp. MDN provides excellent tutorials and documentation, while the project assignments on freeCodeCamp put that knowledge into action.

I tend to move at a snail's pace - an exaggeration, I move even slower - so progress here probably won't be astronomical :)

Update (June 17, 2020): I put the brakes on learning web development a few months ago. The resources mentioned above, however, are still valid and highly recommended for those with an interest in the field.
